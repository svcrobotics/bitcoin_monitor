# TANSA

**Bitcoin, expliqué par les données.**

Tansa observe Bitcoin en temps réel, vérifie les faits directement sur la blockchain et répond à des questions simples avec des données vérifiables.

Exemples :

* Pourquoi le Bitcoin baisse aujourd'hui ?
* Les whales accumulent-elles ?
* Les exchanges reçoivent-ils plus de BTC qu'ils n'en envoient ?
* Le réseau est-il en bonne santé ?

## Notre principe

Tansa ne donne pas d'opinion.

Chaque réponse repose sur des faits observés, certifiés et traçables.

## Architecture

```
Bitcoin Core
      │
   Layer1
      │
   Cluster
      │
 ActorProfile
      │
 ActorLabels
      │
 Questions / Réponses IA
```

Chaque module ajoute de l'information sans modifier les faits des modules précédents.

## Objectif

Construire le moteur de compréhension de Bitcoin le plus fiable possible.

Les données sont certifiées avant d'être utilisées par l'IA.

## Philosophie

Ne jamais inventer.
